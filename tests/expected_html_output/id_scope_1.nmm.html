§ 1

Section 1

¶ 1.1

The ID (‘PAR:par_default_scope’) of this paragraph has default (global) scope. It may be referred to from anywhere in the document.

¶ 1.2

The ID (‘PAR:par:GBL’) of this paragraph has explicitly global scope.

§ 2

Section 2

¶ 2.1

The ID (‘PAR:par:SEC’) of this paragraph has section scope. We may still refer to ¶ 1.2, by using ‘[PAR:par:GBL]’.
