¶ 2.3

DEFINITION

We say that Γ ⊨ φ just in case every model of Γ is a model of φ.

¶ 2.4
DEFINITION

Provability

We say that Γ ⊢ φ just in case there is a proof from Γ to φ.

¶ 2.5

FACT

For every sentence φ and set of sentences Γ, if Γ ⊨ φ then Γ ⊢ φ.
